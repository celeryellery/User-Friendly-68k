About


What is this website for?
This website is a work-in-progress. It was created by students as a compilation of existing knowledge about the 68k processor. Learning to program in the 68k's assembly language can be overwhelming, since the available resources are scattered and very technical. The goal of this website it to provide easy access to the most basic information about the 68k, so that students and novice programmers can begin projects as quickly as possible.


What is the 68000?
The 68000, also known as the 68k, is a microprocessor first developed by Motorola in the 1970's. This processor has been used in a wide variety of embedded systems, as well as for educational purposes.


What is an opcode?
An opcode is a short sequence of letters that specifies the command a processor will execute. The set of opcodes that a particular processor can understand make up its instruction set. Knowledge of a processor's opcodes and how to use them is essential to programming the processor in assembly language.


How do I add an opcode to the table?
Since this website is still incomplete, our table is missing many opcodes. Please contribute to this project by researching and adding more opcodes. Visit our homepage and select the "+ Add an Opcode" button in the upper right-hand corner, and enter the relevant information. Please refer to our Further Reading page to find sites you can use to research opcode information.
